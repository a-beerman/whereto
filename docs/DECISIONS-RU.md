# Decisions log (RU)

## 2025-12-15 — B2C-first + Offline catalog
**Решение:** строим B2C-first продукт, каталог мест наполняем офлайн (Google Places → наша БД по городам).  
**Почему:** скорость UX, контроль стоимости/rate limits, возможность overrides/курирования, независимость от upstream.

## 2025-12-15 — Scope Phase 1
**In scope:** discovery/search → карточка места → save/share; бот — основной канал.  
**Out of scope:** меню/товары, бронирования, заказы/оплаты, B2B кабинет.

## 2025-12-15 — Phase 2
**Решение:** B2B (онбординг заведений, availability/booking, меню) переносим после M2.
